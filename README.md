<h3>Applying Redux into React Hooks</h3>

- Install Redux, React-redux
- Create Store
  - Initial state
  - Reducer function
  - Store with createStore
- Provider off top level component to pass on store
- State with useSelector
- Dispatch actions with useDispatch Hook
